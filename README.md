# Will the Customer Accept the Coupon?

## Overview
This repository contains the Module 5 assignment notebook and supporting data files for a coupon analysis project. Based on the repository structure, the project includes a Jupyter Notebook named `UCBEHass-PCMLAI-MOD5-Coupon-Analysis.ipynb` and a `data/` folder for input datasets. The goal of the project is to analyze coupon-related data, explore patterns, and build clear data-driven insights using Python in a notebook environment.

## Repository Structure
```text
UCBEHass-PCMLAI-MOD5/
├── data/
└── UCBEHass-PCMLAI-MOD5-Coupon-Analysis.ipynb
```

## Project Objectives
- Load and explore the coupon dataset
- Perform data cleaning and preprocessing
- Analyze customer and coupon behavior patterns
- Visualize important trends and relationships
- Summarize findings and business insights

## Tools and Libraries
This notebook is expected to use a standard Python data analysis stack such as:
- Python 3.x
- Jupyter Notebook
- pandas
- numpy
- matplotlib
- seaborn

## Getting Started
### 1. Clone the repository
```bash
git clone https://github.com/rsatsangi/UCBEHass-PCMLAI-MOD5.git
cd UCBEHass-PCMLAI-MOD5
```

### 2. Create and activate a virtual environment (optional)
```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

### 3. Install dependencies
```bash
pip install jupyter pandas numpy matplotlib seaborn
```

### 4. Launch Jupyter Notebook
```bash
jupyter notebook
```
Then open:
`UCBEHass-PCMLAI-MOD5-Coupon-Analysis.ipynb`
